调校Vista

Posted on 2008-02-26

早上在西行资讯看到一篇Vista调校的文章, 说的都是我还不知道的方法, 感觉很有用, 于是摘来备用:)

1、UAC：其实，对付UAC不仅仅是关闭一种方法，除了开启全功能UAC、关闭UAC外，我们还可以即开启UAC（利用其安全功能）又可以让其在”安静”模式下工作。我们需要借助TweakUAC工具，可以点击这里下载。

2、更快的磁盘性能：仅针对SATA磁盘有效。再向磁盘写入前，Vista会使用内存作为高速缓存，风险就是突然断电会致使这部分数据丢失。

3、加速网络文件性能：可能是系统设计原因致使网络文件操作速度降低，可以使用管理员权限运行”cmd”，执行
netsh int tcp set global autotuninglevel=disabled
命令，禁用Vista的”Auto Tuning”功能，开启可以使用这个命令：
netsh int tcp set global autotuninglevel=normal
